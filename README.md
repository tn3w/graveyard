# graveyard
Retired side projects: experiments, prototypes, and abandoned ideas. Archived for reference and possible future revival.

## Projects in this archive

Below is a quick overview of each directory contained in this repository. The original source, README, or core file was consulted for summaries.

- **ai/** – Internal prompts and specification system (SPEC.md/IMPLEMENTATION_PLAN.md) used to manage AI development tasks.
- **antigravity-bin/** – Arch Linux package for Google Antigravity IDE; contains AUR build scripts and update tooling.
- **Clouded/** – Rust-based GitHub deployment platform with web UI, captcha, and user authentication services.
- **CipherChat/** – Console application for secure anonymous chatting over Tor, including bridge management.
- **chromakey/** – Flask authentication library tailored for presentations and streaming (visual/token auth).
- **editpy/** – Terminal-based code editor and hex viewer written in Python using curses.
- **flask_AuthGenius/** – Advanced Flask account and authorization system for web apps.
- **IAm/** – Rust web app providing authentication and CAPTCHA services with WebAuthn support.
- **IPApi-PY/** – Deprecated Python-powered IP information API (replaced by a Rust reimplementation).
- **MouseMovement/** – Browser extension to record cursor movement for AI training (privacy warning included).
- **PoF/** – Flask extension to protect routes from bots and DDoS (“Proof of Fun” / protective middleware).
- **pq-chat-app/** – End‑to‑end encrypted chat using post‑quantum cryptography; Rust server and WASM client.
- **proxycall/** – Python tool for testing rate limiting through proxy rotation (with ethical usage notice).
- **PyVault/** – Python utility for secure file operations: encryption, deletion, signature, etc.
- **repo/** – Personal portfolio web app in Rust, similar to GitHub with Markdown rendering.
- **rusty-loadbalancing/** – Rust load‑balancer component using Redis for least‑connections routing.
- **statuspage/** – Rust web frontend for `statussentinel`; real‑time service status dashboard.
- **statussentinel/** – Rust service monitoring tool with PostgreSQL backend for web and Minecraft servers.
- **SillyPython/** – CLI that surrounds Python scripts with junk code to obscure them.
- **Tunnels/** – HTML/CSS design mockup for a peer‑to‑peer tunnelling interface.
- **cltunnel/** – Python prototype client implementing a Cloudflare Argo-style tunnel (QUIC).
- **vigilus/** - Collection of browser side anti bot measures.
- **crua/** - Fast crawler detection and browser parsing for Python.

> 🗂️ _Many of these projects are proofs-of-concept or abandoned; they remain here for historical reference and potential reuse._
